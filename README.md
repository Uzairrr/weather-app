# Weather App

A live weather dashboard built with vanilla HTML, CSS, and JavaScript.

## Features
- Search any city in the world
- Live data from the [Open-Meteo API](https://open-meteo.com/) — no API key required
- Current temperature, feels like, humidity, and wind speed
- 7-day daily forecast with weather icons

## Usage
Open `index.html` in any browser. No install or build step needed.
